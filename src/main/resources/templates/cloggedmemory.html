Clogged Memory

This is a memory-based game. Hope you have done your readings well!!!!!!!
Now is the time to sharpen and test your memory. You only have 60 seconds to play the game.
For you to be able to play, click on START. After clicking the START, the timer will start ticking.
You must try flipping two cards matching in a row until all the cards are matched within 60 seconds. For example, the cards to match, both the flipped cards should have the same content on it such as Egg Shells - Egg Shells, Toilet Paper - Toilet Paper and so on.
If the flipped card is not a match i.e. Toilet Paper - Egg Shell, then the cards are automatically flipped back.
You need to remember which card is placed where so that in the next turn you can flip the correct cards to make a match.
If you match all the cards within 60 seconds, then you WIN.
If you are not able to make a match within 60 seconds, then you LOSE and the Game Ends.
